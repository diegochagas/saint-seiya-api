What is this?

This website provides a RESTful API interface to highly detailed objects built from thousands of lines of data related to Saint Seiya. Using this website, you can consume information on Saint Seiya, their characters, cloths, classes and more.

What is an API?

An API (Application Programming Interface) is a contract that allow developers to interact with an application through a set of interfaces. In this case, the application is a database of thousands of Saint-Seiya-related objects, and the interfaces are URL links. A RESTful API is an API that conforms to a set of loose conventions based on HTTP verbs, errors, and hyperlinks.

What's the technology stack?

The website were built using Angular framework and CSV files database to store the data. The API were build with Node and Express, that were used to expose the data through a RESTful API, that converted CSV files to serve static JSON files in a fully backwards compatible manner. The images was storage on Firebase.
