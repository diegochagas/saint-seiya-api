Introduction

With Saint Seiya API you can develop applications for Saint Seiya fans with only a few lines of code. You will see bellow descriptions about the things you can do with this API.

Characters

Retrieve data about the characters of Saint Seiya

Through Saint Seiya API you can get the personal data about the characters from the anime, manga and other midias, as their age, family, blood type and other informations.

Classes

Retrieve data about the classes of Saint Seiya

Here you can retrieve informations about several types of classes of Saint Seiya, like Saints cloths, Specters ranks, Marines techniques, etc.

Debuts

Retrieve data about the debuts of Saint Seiya

In this section you can be aware of the types of midias that Saint Seiya have, as manga, anime, etc. Besides that, you can learn the names of the debuts, and their release dates.
